# AI Background Image Prompts

Use these prompts to generate page backgrounds that match each route context.

The project currently includes SVG placeholder backgrounds with these filenames. Replace each file with your generated image using the same filename to apply it instantly.

## Generation Spec

- Aspect ratio: 16:9
- Recommended size: 2048x1152 (or 1920x1080 minimum)
- Style: cinematic, editorial, high-detail, clean composition
- Color direction: cyan accents, deep navy shadows, subtle violet highlights, low visual noise
- No text, no logos, no watermarks, no UI elements
- Keep center and upper-middle region visually calmer so headings remain readable
